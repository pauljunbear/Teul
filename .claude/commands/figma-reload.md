Instructions for reloading the plugin in Figma after making changes.

## Quick Reload
1. Press Cmd+Option+P in Figma to rerun the last plugin

## Full Reload (if quick reload doesn't pick up changes)
1. Close the plugin window if open
2. Plugins → Development → Teul → Run

## Re-import (if manifest.json changed)
1. Plugins → Development → Import plugin from manifest
2. Navigate to: /Users/paul/Desktop/Wado Sanzo Color/manifest.json
3. Select and import
